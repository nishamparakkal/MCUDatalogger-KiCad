# MCU Datalogger

## Overview

A microcontroller-based data logger designed in KiCad.

The design uses an ATmega328P microcontroller, DS1337 real-time clock, and external EEPROM memory for time-stamped data storage.

## Features

- ATmega328P-AU microcontroller
- DS1337 Real-Time Clock (RTC)
- Dual 24LC1025 EEPROM memory chips
- External crystal oscillators
- Status indicator LEDs
- I2C communication bus
- Expansion pin headers
- Two-layer PCB design
- ERC and DRC verified
- Manufacturing-ready Gerber files

## Tools Used
- KiCad 10
- Git
